Setup Questions

Instructions

In this section, you can define a list of questions for users who will be importing your settings. During the import process, DART will copy the user's answers to the fields you specify.

For example, DART does not export login names and passwords. If you export a Storage Service record for others to import, you can add questions here asking the user for their credentials to access that Storage Service.

You may ask "What is your password for the sftp server?", and tell DART to copy the user's response into the password field of the Storage Service called "SFTP demo server".
