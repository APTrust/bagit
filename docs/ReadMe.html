Welcome to EasyStore

EasyStore provides a simple drag-and-drop interface for packaging files into BagIt format and uploading them to APTrust.

Getting Started on Windows

  1. Run the installer, EasyStore Setup.exe.
  2. Copy the file apt_create_bag.exe to a folder on your computer.
  3. Start the EasyStore application.
  4. Choose Settings > Application Settings from the Easy Store menu.
  5. Click the setting called Path to Bagger
  6. In the Value field, type in the path where you saved the apt_create_bag application. E.g.: C:\Users\josie\bin\apt_create_bag.exe
  7. Click Save

Configuration

Someone from APTrust will help you configure your APTrust BagIt Profile and the Storage Service settings for uploading bags to APTrust's demo and production repositories.
